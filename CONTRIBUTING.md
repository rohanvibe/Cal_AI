# Contributing to Cal AI

First off, thank you for considering contributing to Cal AI! It's people like you that make Cal AI such a great tool.

## Code of Conduct
This project and everyone participating in it is governed by our Code of Conduct. By participating, you are expected to uphold this code.

## How Can I Contribute?
- **Reporting Bugs**: Open an issue with a clear description of the bug and steps to reproduce.
- **Suggesting Features**: We love new ideas! Explain the "Why" and "How" of your feature request.
- **Pull Requests**:
  1. Fork the repo.
  2. Create a new branch (`feat/awesome-feature`).
  3. Commit your changes.
  4. Push to your branch.
  5. Open a Pull Request.

## Development Setup
Refer to the [README.md](README.md) for local setup instructions.

## Style Guide
- Use TypeScript for all new files.
- Follow the existing folder structure.
- Run `npm run lint` before submitting a PR.
- Use meaningful commit messages.
